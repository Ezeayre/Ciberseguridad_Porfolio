# Log Files Guide - Entender Logs desde Cero

Una guía para entender qué son los logs, cómo leerlos y por qué importan en ciberseguridad.

## ¿Para quién es?

Para cualquiera que vea un archivo de log y no entienda qué está pasando. Principiantes, juniors, o gente curiosa.

## ¿Qué vas a entender?

- Qué es un log y para qué sirve
- Cómo está estructurado un log
- Qué información puedes sacar de un log
- Cómo buscar problemas en logs
- Por qué los analistas de seguridad pasan horas leyendo logs

## Contenido

1. **¿Qué es un log?** - Concepto básico
2. **Estructura de un log** - Desglose línea por línea
3. **Tipos de logs** - syslog, auth, Apache, etc.
4. **Cómo leer un log** - Herramientas y técnicas
5. **Ejercicio práctico** - Encuentra el problema

## Cómo usar esta guía

Lee en orden. Cada sección tiene ejemplos reales que puedes ver en tu terminal.

No necesitas experiencia previa. Si no sabes qué es un log, este es el lugar correcto.
