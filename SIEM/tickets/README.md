# 🎫 Guía de Tickets en SOC

Qué son los tickets, cómo se usan, y cómo se llenan en un Security Operations Center.

## ¿Para quién es?

- Analistas SOC que recién comienzan
- Cualquiera que necesite entender el sistema de tickets
- Equipos de IT, Help Desk y operaciones

## ¿Qué incluye?

- **¿Qué es un ticket?** - Definición simple
- **Estructura** - Qué campos tiene un ticket
- **Cómo se llena** - Paso a paso
- **Dónde se usan** - Departamentos y casos
- **Ciclo de vida** - Estados por los que pasa
- **Ejemplo real** - Ticket de phishing completo

## Herramientas comunes

- Jira
- ServiceNow
- Azure DevOps
- Zendesk
- Redmine
