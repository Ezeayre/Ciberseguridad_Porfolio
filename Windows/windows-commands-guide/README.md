Una guía interactiva de comandos Windows (CMD y PowerShell) organizada por nivel de dificultad. Diseño moderno con glassmorphism, filtros dinámicos y cards expandibles.

---

## ✨ Características

- **30 comandos esenciales** organizados en 3 niveles:
  - 📚 **Básico** (10 comandos)
  - 🔍 **Intermedio** (10 comandos)
  - 🔐 **Avanzado** (10 comandos)

- **Interfaz moderna y responsive**
  - Diseño con glassmorphism y gradientes
  - Cards expandibles con animaciones suaves
  - Filtros por nivel de dificultad
  - Búsqueda en tiempo real

- **Información completa por comando**
  - Sintaxis clara
  - Ejemplos prácticos
  - Opciones principales
  - Advertencias de seguridad
  - Cuándo usar cada comando

- **Secciones adicionales**
  - 💡 Tips importantes para CMD/PowerShell
  - 📖 Próximos pasos para seguir aprendiendo

---

## 🚀 Cómo usar

1. **Navega por niveles** usando los botones de filtro
2. **Haz clic en una card** para expandir y ver detalles
3. **Usa la búsqueda** para encontrar comandos específicos
4. **Practica** los comandos en tu terminal

---

## 📚 Comandos incluidos

### Nivel Básico
- `ipconfig` - Configuración de red
- `ping` - Verificar conectividad
- `tasklist` - Lista procesos
- `taskkill` - Termina procesos
- `dir` - Lista archivos
- `cd` - Cambia directorio
- `copy` - Copia archivos
- `del` - Elimina archivos
- `cls` - Limpia pantalla
- `echo` - Imprime texto

### Nivel Intermedio
- `netstat` - Conexiones de red
- `Get-Process` - Procesos (PowerShell)
- `systeminfo` - Info del sistema
- `wmic` - WMI command-line
- `gpupdate` - Políticas de grupo
- `nslookup` - Resolución DNS
- `tracert` - Rastreo de ruta
- `netsh advfirewall` - Firewall
- `sfc` - System File Checker
- `chkdsk` - Verificación de disco

### Nivel Avanzado
- `Get-EventLog` - Logs del sistema
- `icacls` - Permisos de archivos
- `auditpol` - Auditoría de seguridad
- `reg` - Registro de Windows
- `robocopy` - Copia robusta
- `cipher` - Encriptación EFS
- `certutil` - Certificados y hashes
- `whoami` - Usuario y permisos
- `runas` - Ejecutar como otro usuario
- `PowerShell pipes` - Combinar comandos
