🪟 Windows Commands Guide

Cheat Sheet Práctico Interactivo

📚 Nivel Básico

ipconfig ▼

Muestra la configuración de red (IP, puerta de enlace, DNS)

Sintaxis:
ipconfig [opciones]
Ejemplos:
ipconfig # Muestra IP actual

ipconfig /all # Muestra detalles completos

ipconfig /release # Libera IP

ipconfig /renew # Renueva IP
Opciones principales:
/all = Muestra detalles completos
/release = Libera dirección IP
/renew = Renueva dirección IP
/flushdns = Limpia caché DNS
¿Cuándo lo usas?

Siempre que necesites verificar la configuración de red

ping ▼

Verifica la conectividad con otro host en la red

Sintaxis:
ping
Ejemplos:
ping google.com # Verifica conectividad

ping 192.168.1.1 # Verifica router

ping -t google.com # Ping continuo (Ctrl+C para detener)
¿Cuándo lo usas?

Para verificar si un host está accesible

tasklist ▼

Lista todos los procesos en ejecución

Sintaxis:
tasklist [opciones]
Ejemplos:
tasklist # Muestra todos los procesos

tasklist | findstr chrome # Busca procesos de Chrome

tasklist /v # Información detallada
¿Cuándo lo usas?

Para ver qué aplicaciones están corriendo

taskkill ▼

Termina un proceso en ejecución

Sintaxis:
taskkill /IM /F
Ejemplos:
taskkill /IM chrome.exe /F # Cierra Chrome forzadamente

taskkill /PID 1234 /F # Cierra por ID de proceso

taskkill /F /IM notepad.exe # Fuerza cierre
⚠️ CUIDADO: Requiere permisos de administrador. /F fuerza el cierre sin confirmación.
¿Cuándo lo usas?

Cuando necesitas cerrar una aplicación que no responde

dir ▼

Lista archivos y carpetas de un directorio

Sintaxis:
dir [opciones] [ruta]
Ejemplos:
dir # Lista carpeta actual

dir C:\Users # Lista contenido de Users

dir /s # Búsqueda recursiva

dir /a # Incluye archivos ocultos
Opciones principales:
/s = Búsqueda recursiva en subcarpetas
/a = Muestra archivos ocultos
/w = Formato ancho
/p = Pausa después de cada pantalla
¿Cuándo lo usas?

Constantemente para ver qué hay en carpetas

cd ▼

Cambia de directorio (carpeta)

Sintaxis:
cd
Ejemplos:
cd C:\Windows # Va a carpeta Windows

cd .. # Sube un nivel

cd \ # Va a raíz del disco

cd /d D:\ # Cambia de unidad
¿Cuándo lo usas?

Para navegar por el sistema de archivos

copy ▼

Copia archivos de un lugar a otro

Sintaxis:
copy
Ejemplos:
copy archivo.txt copia.txt # Copia archivo

copy archivo.txt D:\backup\ # Copia a otra carpeta

copy *.txt D:\docs\ # Copia múltiples archivos
¿Cuándo lo usas?

Para duplicar archivos o hacer backups

del ▼

Elimina archivos

Sintaxis:
del [opciones]
Ejemplos:
del archivo.txt # Elimina archivo

del /P archivo.txt # Pide confirmación

del *.tmp # Elimina todos los .tmp
⚠️ CUIDADO: No hay papelera de reciclaje desde CMD. Una vez eliminado, es difícil recuperarlo.
¿Cuándo lo usas?

Cuando necesitas eliminar archivos

cls ▼

Limpia la pantalla de la consola

Sintaxis:
cls
Ejemplos:
cls # Borra todo el contenido
¿Cuándo lo usas?

Para limpiar el desorden en la pantalla

echo ▼

Imprime texto en pantalla

Sintaxis:
echo
Ejemplos:
echo Hola Mundo # Imprime texto

echo %USERNAME% # Imprime variable de usuario

echo texto > archivo.txt # Guarda texto en archivo
¿Cuándo lo usas?

Para imprimir texto o guardar información

🔍 Nivel Intermedio

netstat ▼

Muestra conexiones de red activas y puertos abiertos

Sintaxis:
netstat [opciones]
Ejemplos:
netstat -an # Muestra todas las conexiones

netstat -b # Incluye nombres de procesos

netstat -o # Muestra IDs de proceso
Opciones principales:
-a = Muestra todas las conexiones
-n = Muestra direcciones numéricas
-b = Muestra el ejecutable
-o = Muestra ID de proceso
¿Cuándo lo usas?

Para auditoría de seguridad y diagnóstico de red

Get-Process ▼

Lista procesos (PowerShell)

Sintaxis:
Get-Process [opciones]
Ejemplos:
Get-Process # Muestra todos los procesos

Get-Process chrome # Muestra proceso específico

Get-Process | Where-Object {$_.Memory -gt 100MB}
¿Cuándo lo usas?

PowerShell es más poderoso que tasklist

systeminfo ▼

Muestra información del sistema (SO, RAM, CPU)

Sintaxis:
systeminfo
Ejemplos:
systeminfo # Información completa

systeminfo | findstr /C:"OS Name"
¿Cuándo lo usas?

Para auditoría de hardware y configuración

wmic ▼

Windows Management Instrumentation Command-line

Sintaxis:
wmic list
Ejemplos:
wmic os list brief # Info del SO

wmic logicaldisk list # Info de discos

wmic process list brief # Lista procesos
¿Cuándo lo usas?

Herramienta poderosa para administración

gpupdate ▼

Actualiza las políticas de grupo

Sintaxis:
gpupdate [opciones]
Ejemplos:
gpupdate /force # Actualiza políticas inmediatamente
⚠️ CUIDADO: Requiere permisos de administrador
¿Cuándo lo usas?

Para aplicar cambios de políticas de grupo

nslookup ▼

Resuelve nombres de dominio a IP

Sintaxis:
nslookup
Ejemplos:
nslookup google.com # Encuentra IP de dominio

nslookup 8.8.8.8 # DNS inverso
¿Cuándo lo usas?

Para diagnóstico de DNS

tracert ▼

Rastrea la ruta de paquetes hasta un host

Sintaxis:
tracert
Ejemplos:
tracert google.com # Muestra cada salto

tracert -d google.com # No resuelve nombres
¿Cuándo lo usas?

Para diagnosticar problemas de red

netsh advfirewall ▼

Gestiona Windows Defender Firewall

Sintaxis:
netsh advfirewall
Ejemplos:
netsh advfirewall show allprofiles

netsh advfirewall set allprofiles state on
⚠️ CUIDADO: Requiere permisos de administrador
¿Cuándo lo usas?

Para configurar el firewall

sfc ▼

System File Checker - verifica integridad de archivos

Sintaxis:
sfc /scannow
Ejemplos:
sfc /scannow # Escanea y repara archivos
⚠️ CUIDADO: Requiere permisos de administrador. Puede tardar tiempo.
¿Cuándo lo usas?

Cuando hay problemas con archivos de sistema

chkdsk ▼

Verifica integridad de disco duro

Sintaxis:
chkdsk /F
Ejemplos:
chkdsk C: /F # Verifica y repara unidad C:
⚠️ CUIDADO: Requiere permisos de administrador
¿Cuándo lo usas?

Para verificar errores de disco

🔐 Nivel Avanzado

Get-EventLog ▼

Accede a logs del sistema (PowerShell)

Sintaxis:
Get-EventLog -LogName
Ejemplos:
Get-EventLog -LogName System -Newest 10

Get-EventLog -LogName Security | Where-Object {$_.EventID -eq 4625}
¿Cuándo lo usas?

Crítico para forensics y auditoría

icacls ▼

Gestiona permisos de archivos y carpetas

Sintaxis:
icacls /grant :
Ejemplos:
icacls archivo.txt /grant Admin:F

icacls carpeta /grant Usuario:R
Opciones principales:
F = Full control (control total)
R = Read (lectura)
W = Write (escritura)
M = Modify (modificar)
¿Cuándo lo usas?

Para controlar permisos de archivos

auditpol ▼

Configura auditoría de seguridad

Sintaxis:
auditpol /get /category:*
Ejemplos:
auditpol /get /category:* # Muestra políticas

auditpol /set /subcategory:"Logon" /success:enable
⚠️ CUIDADO: Requiere permisos de administrador
¿Cuándo lo usas?

Para configurar auditoría de seguridad

reg ▼

Accede y modifica el Registro de Windows

Sintaxis:
reg
Ejemplos:
reg query HKLM\Software # Busca en Registro

reg add HKLM\Software\Key /v Value /d Data
⚠️ CUIDADO: ¡PELIGROSO! Modificar el Registro incorrectamente puede dañar el sistema.
¿Cuándo lo usas?

Para configuración avanzada del sistema

robocopy ▼

Copia de archivos robusta y recursiva

Sintaxis:
robocopy [opciones]
Ejemplos:
robocopy C:\Users D:\Backup /S /E

robocopy origen destino /MIR # Espejo exacto
Opciones principales:
/S = Copia subcarpetas (vacías no)
/E = Copia subcarpetas (vacías sí)
/MIR = Espejo (borra en destino si no existe en origen)
¿Cuándo lo usas?

Para backups robustos de carpetas enteras

cipher ▼

Encripta/desencripta archivos (EFS)

Sintaxis:
cipher [opciones]
Ejemplos:
cipher /e C:\datos # Encripta carpeta

cipher /d C:\datos # Desencripta carpeta
¿Cuándo lo usas?

Para proteger archivos sensibles con EFS

certutil ▼

Gestiona certificados y hashes

Sintaxis:
certutil [opciones]
Ejemplos:
certutil -hashfile archivo.exe SHA256

certutil -dump archivo.cer
¿Cuándo lo usas?

Para verificar hashes de archivos

whoami ▼

Muestra el usuario actual y sus permisos

Sintaxis:
whoami [opciones]
Ejemplos:
whoami # Usuario actual

whoami /priv # Muestra privilegios

whoami /groups # Muestra grupos
¿Cuándo lo usas?

Para verificar permisos del usuario

runas ▼

Ejecuta un programa como otro usuario

Sintaxis:
runas /user:
Ejemplos:
runas /user:admin cmd.exe # Abre CMD como admin

runas /user:dominio\usuario programa.exe
⚠️ CUIDADO: Pide contraseña del otro usuario
¿Cuándo lo usas?

Para ejecutar programas con permisos elevados

PowerShell pipes ▼

Combina comandos para operaciones complejas

Sintaxis:
comando1 | comando2 | comando3
Ejemplos:
Get-Process | Where-Object {$_.CPU -gt 10}

Get-EventLog -LogName Security | Select-Object -First 10
¿Cuándo lo usas?

Para operaciones complejas encadenando comandos

💡 Tips Importantes

  • Usa Tab - Autocomplete de comandos y rutas
  • Usa Ctrl+C - Detiene comando en ejecución
  • Usa PowerShell - Más poderoso que CMD
  • Usa Arrow Up - Historial de comandos anteriores
  • Combina comandos - Usa pipes (|) en PowerShell
  • Lee documentación - Usa `Get-Help comando` para aprender

📖 Próximos Pasos

  1. Practica en PowerShell/CMD
  2. Experimenta combinando comandos
  3. Lee la documentación (`Get-Help`)
  4. Usa `tasklist` y `taskkill` para procesos
  5. Administra archivos y carpetas
