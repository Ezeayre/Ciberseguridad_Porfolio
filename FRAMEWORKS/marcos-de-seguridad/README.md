# Guía Interactiva de Marcos de Seguridad

**Autor:** Ezequiel Ayre  
**Fecha:** 2026  
**Archivo:** marcos-de-seguridad.html

## Descripción

Guía interactiva sobre los marcos de seguridad más utilizados en la industria: NIST CSF, ISO 27001 y CIS Controls. Diseñada como recurso educativo para profesionales de ciberseguridad.

## Características

### Contenido Principal (9 Secciones)

1. **¿Qué es un Marco de Seguridad?**
   - Definición y propósito de los marcos de seguridad
   - Analogía práctica para entender su importancia

2. **NIST Cybersecurity Framework (CSF)**
   - Las 5 funciones clave: Identify, Protect, Detect, Respond, Recover
   - Niveles de madurez (1-5)
   - Ventajas y desventajas

3. **ISO/IEC 27001**
   - 14 dominios de seguridad
   - Proceso de certificación formal
   - Ventajas y desventajas

4. **CIS Controls**
   - 20 controles organizados en 3 grupos: Básicos, Esenciales, Avanzados
   - 3 niveles de implementación
   - Enfoque práctico

5. **Comparación: NIST vs ISO 27001 vs CIS**
   - Tabla comparativa detallada
   - Análisis por aspectos clave

6. **¿Cuándo usar cada uno?**
   - Recomendaciones según contexto organizacional
   - Mejor estrategia combinada

7. **Marcos Complementarios**
   - COBIT (Gobernanza IT)
   - ITIL (Operaciones IT)
   - PCI-DSS (Pagos con tarjeta)

8. **Ejemplo Real: Empresa Fintech**
   - Caso práctico de implementación combinada
   - Resultados y roles de cada marco

9. **Implementación Básica**
   - 7 pasos para comenzar
   - Roadmap inicial

### Funcionalidades Interactivas

- **Búsqueda en tiempo real:** Filtra contenido por términos clave
- **Tema claro/oscuro:** Toggle con persistencia en localStorage
- **Secciones expandibles:** Sistema acordeón para cada sección
- **Navegación por sidebar:** Índice lateral con links directos
- **Glosario interactivo:** 16 términos clave en formato acordeón

## Cómo Usar

1. Abra el archivo `marcos-de-seguridad.html` en cualquier navegador web moderno
2. Use el sidebar izquierdo para navegar entre secciones
3. Use la barra de búsqueda para filtrar contenido
4. Haga clic en los encabezados para expandir/colapsar secciones
5. Use el botón Dark/Light para cambiar el tema

## Requisitos Técnicos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- JavaScript habilitado
- Conexión a internet no requerida (funciona offline)
