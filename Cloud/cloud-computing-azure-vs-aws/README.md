# ☁️ Guía de Cloud Computing - AWS vs Azure

> **Tu guía interactiva para dominar la nube desde cero**

---

## 🚀 ¿Qué es esto?

Una guía interactiva y visual que te explica **Cloud Computing** de forma sencilla, comparando los dos gigantes del mercado: **AWS** (Amazon) y **Azure** (Microsoft).

Ideal si estás empezando en el mundo cloud, quieres entender los servicios básicos, o necesitas una referencia rápida para tu trabajo diario.

---

## 🎯 Para quién es

- **Juniors** que recién se inician en cloud
- **Desarrolladores** que quieren entender servicios cloud
- **SysAdmins** que migran de on-premise a la nube
- **Estudiantes** que necesitan una guía clara y práctica
- **Cualquiera** que quiera aprender cloud sin complicaciones

---

## ✨ ¿Qué encontrarás?

### 📚 **Conceptos Fundamentales**
- IaaS, PaaS, SaaS explicados simple
- Regiones, Availability Zones, y por qué importan
- Modelos de pricing (pay-as-you-go)

### 🔶 **AWS - Servicios Principales**
- **EC2**: Máquinas virtuales
- **S3**: Almacenamiento en la nube
- **RDS**: Bases de datos gestionadas
- **Lambda**: Funciones sin servidor
- **IAM**: Control de acceso

### 🔵 **Azure - Servicios Principales**
- **Virtual Machines**: Servidores virtuales
- **Storage**: Almacenamiento masivo
- **SQL Database**: Bases de datos SQL
- **Functions**: Serverless computing
- **Entra ID**: Gestión de identidad

### 🛡️ **Seguridad en Cloud**
- IAM y roles
- Security Groups / Network Security Groups
- Encriptación de datos
- MFA y mejores prácticas

### 📊 **Tabla de Equivalencias**
- AWS ↔ Azure: qué servicio equivale a qué
- Referencia rápida para migrar entre plataformas

### 💻 **Comandos CLI Básicos**
- **AWS CLI**: Comandos para EC2, S3, RDS, IAM
- **Azure CLI**: Comandos para VMs, Storage, SQL, Entra ID
- Cada comando con explicación de qué hace
- Botón de copiar para usarlos directamente

### 💡 **Tips y Buenas Prácticas**
- ✅ Lo que SIEMPRE debes hacer
- ❌ Lo que NUNCA debes hacer
- Consejos para evitar errores comunes

## 📖 Cómo usarla

1. **Abre el archivo HTML** en tu navegador
2. **Navega por secciones** usando los acordeones
3. **Explora las tarjetas** de servicios (click para expandir)
4. **Usa los filtros** para ver solo AWS, Azure o Seguridad
5. **Copia comandos** CLI cuando los necesites
6. **Consulta la tabla** de equivalencias como referencia rápida
