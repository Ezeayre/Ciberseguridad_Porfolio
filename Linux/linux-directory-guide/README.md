# 📁 Linux Directory Structure Guide

¿Alguna vez entraste a una carpeta de Linux y no sabías qué era?

Esta guía te enseña la estructura de directorios de Linux y por qué importan en ciberseguridad.

## ¿Para quién es?

- Juniors que recién empiezan en Linux
- Seniors que quieren una referencia rápida
- Pentesters buscando dónde buscar evidencia
- Sysadmins manteniendo sistemas
- Cualquiera trabajando en seguridad

## ¿Qué incluye?

Una guía práctica de los directorios más importantes: qué contienen, por qué existen, cómo acceder, y qué deberías saber de cada uno.

Sin teoría aburrida. Solo hechos útiles.

## Directorios Cubiertos

- `/bin` - Comandos esenciales
- `/etc` - Configuraciones del sistema
- `/home` - Directorios de usuarios
- `/root` - Home del administrador
- `/var/log` - Logs del sistema
- `/tmp` - Archivos temporales
- `/usr/bin` - Comandos adicionales
- Y más...

## Cómo Usar Esta Guía

Lee cada sección. Prueba los comandos en tu terminal. Entiende por qué cada directorio existe y qué contiene.

Después, cuando trabajes en seguridad, sabrás exactamente dónde buscar.
